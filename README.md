# 📊 ConnectaTel-Analisis-Outliers-y-Patrones

## 📌 Descripción del Proyecto
Este proyecto consiste en un análisis exploratorio de datos enfocado en la **segmentación de usuarios** por grupo de edad y nivel de uso.  

El objetivo es identificar patrones de comportamiento, niveles de engagement y oportunidades de mejora en retención y crecimiento del negocio.

---

## 🎯 Objetivos
- Analizar la distribución de usuarios por edad.
- Evaluar el nivel de uso de la plataforma.
- Detectar posibles riesgos de abandono (churn).
- Generar insights accionables para negocio.

---

## 📂 Dataset
El dataset contiene información sobre:
- Edad del usuario
- Nivel de uso (bajo, medio, alto)
- Variables relacionadas con comportamiento en la plataforma

---

## 🛠️ Herramientas utilizadas
- Python 🐍
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook

---

## 🔎 Análisis realizado
- Limpieza y exploración de datos (EDA)
- Agrupación de usuarios por segmentos
- Visualización de distribuciones
- Identificación de patrones de uso

---

## 📊 Principales Insights
- El segmento **adulto domina la base de usuarios**.
- La mayoría de usuarios presenta **uso medio**.
- Existe una **baja proporción de usuarios de alto uso**.
- El segmento joven está **subrepresentado**, lo que puede limitar el crecimiento.

---

## 💡 Recomendaciones
- Incrementar el engagement para convertir usuarios de uso medio a alto.
- Implementar estrategias de retención para usuarios de bajo uso.
- Diseñar campañas enfocadas en atraer usuarios jóvenes.
- Analizar el comportamiento de usuarios de alto uso para replicarlo.
